### Adding New Clips

Add an entry to `sounds.json`:
```json
{
    "id": "my-clip",
    "label": "My Clip",
    "videoId": "YOUTUBE_VIDEO_ID",
    "start": 10,
    "duration": 5
}
```
- `videoId` ‚Äî the YouTube video ID (the part after `v=`)
- `start` ‚Äî start time in seconds
- `duration` ‚Äî how many seconds to play
